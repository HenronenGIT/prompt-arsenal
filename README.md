# prompt-arsenal

A collection of prompts, rules, and guidelines for AI-assisted development workflows.

## Contents

### `cursor-rules/`
Cursor `.mdc` rule files for common tech stacks — TypeScript, React, Next.js, Node/Express, Python, Tailwind, and general clean code and module design principles.

### `coding-guidelines/`
Markdown guidelines covering architecture philosophy, module design, clean code practices, and common anti-patterns to avoid.

### `codex-automatizations/`
Prompts for automating recurring Codex/Claude tasks: skill creation, security scans, doc link checking, feature idea generation, and pattern scanning.

### `init-guides/`
Step-by-step setup guides for bootstrapping new projects (e.g. TypeScript monorepo with pnpm + Turborepo).
